Dónde se procesa el NSF

GitHub Actions solo descarga el archivo y lo guarda en nsf/. El análisis ocurre en el navegador: la app hace fetch() del NSF, lee los bytes, parsea el header, pasa el archivo al emulador Game_Music_Emu compilado a JavaScript y renderiza audio con Web Audio API.
